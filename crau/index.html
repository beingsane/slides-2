## Crau: web crawler - Necessidade: **arquivar a Web** - Ninguém no Brasil fala **crawl** - (a não ser quando cantamos Linkin Park no karaokê) - (ou na Carreta Furacão, junto com Numb) - `pip install crau`
## Crau - Funcionalidades - Arquiva lista de links com um comando: - `crau archive file.warc.gz URL1 URL2 ...`: - Usa o formato WARC (largamente utilizado nessa área) - Extrai dados do arquivamento: - `crau list file.warc.gz` - `crau extract file.warc.gz URL` - "Toca" o arquivamento no navegador (tipo a [Wayback Machine](https://web.archive.org/) do [Internet Archive](https://archive.org/)): - `crau play file.warc.gz` - Em breve cache storage pro scrapy! - Feito em 2 madrugadas de PyBr
