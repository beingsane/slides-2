## rows: Funcionalidades - Interface única (não importa o formato) - Escrever é tão fácil quanto ler (*pythonic way*) - Vários plugins - CLI - Conversão **automática** de dados - **Eu já fiz os testes** - Test-driven development - Semantic versioning
## Command-Line Interface ### live coding
## Command-Line Interface - `print`: mostra a tabela em formato texto - `convert`: converte entre os formatos suportados - `sum`: agrega tabelas - `join`: cruza dados - `query`: SQL!
